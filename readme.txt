== Description ==

Replaces text and HTML tags in the content.

= Replace =
* Text
* HTML tags

= Filter =
* Post ID
* Post Author
* Post Type
* Post Status
* Categories
* Taxonomies
* Terms
* Date

= Logs =
* Displays the last 100 logs.

= How it works =
[youtube https://youtu.be/bsLnt0ZA_u8]

== Installation ==

1. Upload `replace-contents` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
